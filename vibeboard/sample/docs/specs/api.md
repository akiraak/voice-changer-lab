# API 仕様

## エンドポイント一覧

| メソッド | パス             | 説明                          |
| -------- | ---------------- | ----------------------------- |
| GET      | `/notes`         | メモ一覧                      |
| GET      | `/notes/:id`     | メモ詳細                      |
| POST     | `/notes`         | 新規作成                      |
| PUT      | `/notes/:id`     | 更新（mtime 楽観ロック）      |
| DELETE   | `/notes/:id`     | ソフト削除                    |

## レスポンス共通形式

```json
{ "success": true, "data": {}, "error": null }
```

## エラーコード

- `400`: リクエストのバリデーション失敗
- `404`: リソースが存在しない
- `409`: 楽観ロック失敗（外部で先に更新されていた）
- `500`: サーバ内部エラー
