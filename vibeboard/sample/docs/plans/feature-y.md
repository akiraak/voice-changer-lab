# 古いメモの自動アーカイブ

## 目的・背景

90 日以上更新されていないメモを自動的にアーカイブして
一覧から外したい（手動アーカイブはあるが運用されていない）。

## 対応方針

cron で日次バッチを動かし、`updated_at < now() - 90d` を一括で
`archived = true` にする。アーカイブ前にユーザへ通知メールを送る。

## 影響範囲

- worker に新規ジョブ 1 本
- 通知メールのテンプレ 1 種類

## テスト方針

- 境界値（89 日 / 90 日 / 91 日）で挙動確認
- 通知メールの dry-run モードで本文を確認
