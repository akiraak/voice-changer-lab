# タグ絞り込み機能

## 目的・背景

メモ件数が増えると一覧から目的のメモを探すのが難しい。
タグを付けて絞り込めるようにしたい。

## 対応方針

- `notes` テーブルに `tags JSON` 列を追加
- 本文中の `#tagname` パターンを保存時に自動抽出
- 一覧 API に `?tags=foo,bar` を追加（AND 検索）

## 影響範囲

- DB マイグレーション 1 本
- API: `GET /notes` のクエリパラメータ拡張
- フロント: 一覧ページ・タグ入力 UI

## テスト方針

- 単体: タグ抽出ロジックのスナップショットテスト
- 統合: AND 検索で期待件数が返ることを確認
